Execution Trace

1. Initialize Open and Closed Sets
2. Add Start Node to Open Set
3. While Open Set is not empty:
4. curr = Node in Open with lowest F
5. If curr == Target: Return Path
6. Move curr to Closed Set
7. For each neighbor of curr:
8. If Wall or Closed: continue
9. tempG = curr.G + 1
10. If tempG < neighbor.G or not in Open:
neighbor.G = tempG
neighbor.H = ManhattanDist(neighbor, Target)
neighbor.F = neighbor.G + neighbor.H
neighbor.parent = curr
If not in Open: Add to Open

Theory

A* Search is an informed search algorithm. Unlike Dijkstra's, which expands uniformly in all directions, A* uses a heuristic to guide the search aggressively toward the target.

  • G Cost Distance from Start node.
  • H Cost Heuristic (Manhattan) distance to Target.
  • F Cost G + H. The algorithm always evaluates the lowest F score next.

Complexity

Time Complexity O(b^d)
Space Complexity O(b^d)

$b$ is the branching factor (4) and $d$ is the depth of the solution. The heuristic severely optimizes practical runtimes over raw BFS.
