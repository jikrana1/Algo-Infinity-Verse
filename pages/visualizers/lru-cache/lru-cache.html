Theory

An LRU Cache evicts the least recently used item when it reaches capacity. It achieves exactly O(1) time complexity for all operations by combining two structures:

  • Hash Map Provides O(1) lookups pointing directly to memory addresses.
  • Linked List A Doubly Linked List allows O(1) removals and insertions to maintain usage order without shifting arrays.
