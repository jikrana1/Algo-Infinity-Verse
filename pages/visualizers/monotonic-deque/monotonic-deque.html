Theory

Finding the maximum in a sliding window naively takes O(N×k) time. We can optimize this to O(N) using a Monotonic Deque.

  • Out of Bounds Remove indices from the front that fall outside the window.
  • Monotonicity Remove smaller elements from the back because they can never be the maximum if a larger element arrives after them.

Complexity

Time Complexity O(N)
Space Complexity O(k)

Every element is pushed and popped at most once, making it strictly O(N) linear time.
