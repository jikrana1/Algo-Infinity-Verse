Theory

The KMP Algorithm searches for occurrences of a "word" within a main "text string" in strictly linear time. It does this by preprocessing the pattern to avoid re-evaluating known matching prefixes.

  • LPS Array Tracks the longest proper prefix that is also a suffix.
  • Fallback On a mismatch, `j` jumps back to `lps[j-1]`, safely shifting the pattern forward without resetting the text pointer `i`.

Complexity

Time Complexity O(N + M)
Space (LPS) O(M)

N = Text Length, M = Pattern Length. The text pointer `i` never moves backwards, guaranteeing O(N) execution.
