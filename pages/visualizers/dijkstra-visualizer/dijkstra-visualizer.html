How Dijkstra's Algorithm Works

1. The Setup

Assign a distance of 0 to the start node and ∞ to all other nodes. Place the start node into a Priority Queue.

2. The Greedy Choice

Extract the node with the minimum distance from the Priority Queue. This node is now considered "settled" – we have found its absolute shortest path.

3. Edge Relaxation

For every neighbor of the extracted node, calculate the distance through the current node. If this new distance is less than the neighbor's known distance, update it and push it to the queue.
