Theory

Edit Distance finds the minimum operations required to convert String 1 into String 2. It evaluates overlapping subproblems using 2D Tabulation.

  • Insert Look Left: dp[i][j-1]
  • Delete Look Up: dp[i-1][j]
  • Replace Look Diag: dp[i-1][j-1]
